# Sistem Pembayaran OOP (PHP)

## 📌 Deskripsi
Project ini merupakan implementasi konsep Object-Oriented Programming (OOP) menggunakan PHP.

Konsep yang digunakan:
- Abstract Class
- Interface
- Inheritance
- Polymorphism

---

## 💳 Metode Pembayaran
- Transfer Bank
- E-Wallet
- QRIS

---

## ⚙️ Struktur Program
- Pembayaran (Abstract Class)
- Cetak (Interface)
- TransferBank (Class turunan)
- Ewallet (Class turunan)
- QRIS (Class turunan)

---

## ▶️ Cara Menjalankan
1. Simpan project di folder htdocs (XAMPP)
2. Jalankan Apache
3. Buka browser
4. Akses:
   http://localhost/pembayaran-oop/index.php
